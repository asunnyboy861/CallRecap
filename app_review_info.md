# App Review Information

## Demo Account for Apple Review

### App Functionality
CallRecap is a call recording and AI transcription app. All processing is on-device (Whisper.cpp + Apple Intelligence/local rules). No server account or API key is required for core functionality.

### Subscription Testing
- **Product IDs**:
  - Monthly: `com.zzoutuo.CallRecap.monthly` ($3.99/mo)
  - Yearly: `com.zzoutuo.CallRecap.yearly` ($29.99/yr, 7-day free trial)
  - Lifetime: `com.zzoutuo.CallRecap.lifetime` ($79.99 one-time)

### How to Test
1. Launch app → Complete onboarding (3 pages + permissions)
2. Tap the red record button on the Recordings tab to start a manual recording
3. Tap stop → Recording is saved and auto-transcribed
4. Tap a recording → View Summary / Transcript / Play tabs
5. Settings tab → Configure audio quality, Whisper model, biometric lock
6. Paywall accessible from Settings → "Upgrade to Pro"

### Free Tier Limits
- 5 transcriptions per month
- No AI summaries
- No action items / Reminders integration
- No export (PDF/SRT/TXT)
- No Face ID lock
