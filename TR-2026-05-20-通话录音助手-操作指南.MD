### 1.3 最终命名：CallRecap

**选择理由**：

#### ASO搜索优化维度
- "Call" 是App Store搜索量最高的关键词之一（月搜索量>500K）
- "Recap" 暗示"回顾/总结"，与AI摘要功能完美匹配
- 组合词 "CallRecap" 在App Store中无直接竞品占用
- Subtitle "AI Call Recorder & Smart Notes" 覆盖所有核心搜索词：call recorder, AI, notes, transcription

#### 品牌记忆度维度（最关键）
- **8个字母**，2个音节，极简极短
- "Recap" 是英语母语者日常高频词汇，零认知成本
- **动词化潜力**："Let me CallRecap that" → 自然口语传播
- **对比竞品**：TapeACall（3音节）、Cube ACR（无意义）、Rev Call Recorder（太长）
- 首字母C和R都是硬辅音，发音有力，记忆深刻

#### 功能描述性维度
- "Call" = 通话，明确场景
- "Recap" = 回顾/总结，暗示录音+AI摘要双重功能
- 用户看到名字立即理解：这是一个帮你回顾通话内容的工具
- 比纯"Recorder"更高级——不只是录音，而是智能回顾

#### 情感共鸣维度
- "Recap" 暗示"轻松回顾"，消除"录音很麻烦"的心理障碍
- 给用户掌控感："我的通话，我来回顾"
- 与"害怕忘记重要信息"的痛点直接共鸣

#### 国际化适配维度
- "Call" 和 "Recap" 是全球通用英语词汇
- 无文化歧义，无负面联想
- 在非英语国家（德、法、日、韩）"Call" 已是外来语，"Recap" 在商务场景广泛使用

## 2. 痛点研究与客户需求分析

### 2.1 核心痛点汇总（基于Reddit/Apple论坛/App Store评论深度分析）

#### 痛点1：iOS 18原生通话录音几乎无用（钻石级 💎）

**用户原话**：
> "A recorder that announces 'this call will be recorded' is almost useless, as it doesn't serve the main use cases. People's behavior changes when they know they are being recorded." — Reddit r/ios (高赞帖)

> "I would like to use the recorder to have evidence in cases of scams and fraud, threats, in short, in case I am a victim of a crime." — 同帖

**痛点分析**：
- iOS 18原生录音会自动播报"此通话将被录音"，对方立刻知道
- 在单方同意地区（美国38个州），用户不需要告知对方
- 播报通知改变了对方行为，使录音失去意义
- **无法关闭播报通知**

#### 痛点2：现有第三方App全部是订阅制，价格离谱（钻石级 💎）

**用户原话**：
> "There are no one-time purchases, they are all subscription-based. And I think it's absurd to pay a monthly fee for a damn app like this." — Reddit r/ios

> "DO NOT download this app. $70 for no reason." — Cube ACR App Store评论

**价格对比**：
| App | 定价 | 年费 | 用户评价 |
|-----|------|------|---------|
| TapeACall | $10.99/周 | $571.48/年 | "horrible persistent buzzing" |
| Cube ACR | $9.99/月 | $119.88/年 | "doesn't work at all" |
| Automatic Call Recorder | $6.99/周 | $363.48/年 | "recording only silence" |
| Call Recorder iCall | $7.99/周 | $415.48/年 | "very disappointed" |

**痛点分析**：
- 所有主流App都采用周订阅陷阱（$6.99-$10.99/周）
- 年费高达$120-$571，远超用户心理预期
- 用户强烈渴望一次性购买选项

#### 痛点3：三方通话方式极其笨拙（钻石级 💎）

**用户原话**：
> "You have to put them on speaker and then merge, which then shows up on their side as being conference called... which defeats the entire purpose." — App Store评论

> "You have to answer the call, then go and call another number to record then merge the calls so you don't even get the first half of the conversation." — Cube ACR评论

**痛点分析**：
- 现有App使用三方通话（3-way calling）方式录音
- 步骤繁琐：接听→打开App→拨打录音号码→合并通话
- 丢失通话开头的关键内容
- 部分运营商不支持三方通话
- 对方看到"会议通话"，暴露录音意图

#### 痛点4：录音质量差/经常失败（金级 🥇）

**用户原话**：
> "When I checked the recording, it had stopped at 55 minutes. This was an important call where I needed to transcribe notes afterwards." — ACR MyCalls评论

> "Very disappointed... When it came to receiving the call, I set the application to record for more than 30 minutes. When I hung up and checked the audio, I only heard very distorted noise and no sound." — App Store评论

**痛点分析**：
- 录音中途停止，无任何提示
- 音频失真、只有噪音
- 录到的是静音
- 丢失重要录音无法恢复

#### 痛点5：录音文件管理混乱（金级 🥇）

**用户原话**：
> "I was horrified when the screen came back with the one I selected still there and the one I needed gone!!" — ACR MyCalls评论

> "Stopped allowing me access to my recordings. When you press the 'tape' button, the app just exits with no error msg." — PhoneCallRecorder评论

**痛点分析**：
- 误删无法恢复（无回收站）
- App崩溃后录音丢失
- 无法按联系人/日期有效分类
- 无法搜索录音内容

#### 痛点6：缺乏AI转录和摘要功能（银级 🥈）

**用户原话**：
> "I just want it to record everything and save it to my device. Is it really that complicated?" — Reddit r/ios

**痛点分析**：
- iOS 18原生仅提供基础转录，无AI摘要
- 第三方App转录需额外付费（Rev: $0.25/分钟）
- 用户需要手动整理通话要点
- 缺乏行动项提取功能

#### 痛点7：地区限制严重（银级 🥈）

**用户原话**：
> "Call recording is not currently available in Azerbaijan, Bahrain, Egypt, the European Union, Iran, Iraq, Jordan, Kuwait, Morocco, Nigeria, Oman, Pakistan, Qatar, Russia, Saudi Arabia, South Africa, Turkey, United Arab Emirates, and Yemen." — Apple Support

**痛点分析**：
- iOS 18原生录音在EU等地区不可用
- 用户跨国旅行时功能失效
- 需要一个不受地区限制的解决方案

## 4. GitHub可二次开发项目研究

### 4.1 核心参考项目

#### 项目1：TrollRecorder（⭐2000+ Stars）
- **地址**：https://github.com/Lessica/TrollRecorder
- **技术栈**：Objective-C / C / Swift
- **核心价值**：iOS上最强大的通话录音项目，支持CallKit录音、微信录音、系统音频录制
- **关键特性**：
  - 电话与FaceTime录音
  - CallKit第三方App录音
  - 微信和其他App录音
  - 系统音频录制
  - 悬浮球状态指示
  - iCloud云归档
  - 多种音频格式（m4a/caf/wav）
  - 分离通道录音
  - 回收站功能
  - Touch ID/Face ID保护
- **二次开发价值**：⭐⭐⭐⭐⭐（核心录音引擎参考）
- **限制**：需要TrollStore/越狱，不能直接上架App Store；但其CallKit录音方案和音频处理逻辑可作为重要参考

#### 项目2：WhisperOnDevice-iOS（MIT License）
- **地址**：https://github.com/dsngeu/WhisperOnDevice-iOS
- **技术栈**：Swift + whisper.cpp
- **核心价值**：在iOS设备上完全离线运行Whisper语音转文字
- **关键特性**：
  - 本地运行whisper.cpp
  - 支持ggml-base.en.bin和ggml-small.en.bin模型
  - 包含音频录制功能
  - 纯Swift实现
- **二次开发价值**：⭐⭐⭐⭐⭐（本地转录引擎核心）

#### 项目3：WhisperCore（Swift Package）
- **地址**：https://github.com/EberronBruce/WhisperCore
- **技术栈**：Swift Package + whisper.xcframeworks
- **核心价值**：封装Whisper为Swift Package，可直接集成到Xcode项目
- **二次开发价值**：⭐⭐⭐⭐（快速集成方案）

#### 项目4：local-whisper（活跃维护）
- **地址**：https://github.com/gabrimatic/local-whisper
- **技术栈**：Swift + whisper.cpp + 多平台
- **核心价值**：离线语音听写、转录、清理和TTS，支持macOS/iOS/Android
- **二次开发价值**：⭐⭐⭐⭐（多平台转录方案参考）

#### 项目5：WhisperCppKit（Swift Package）
- **地址**：https://github.com/Justmalhar/WhisperCppKit
- **技术栈**：Swift Package + whisper.cpp预编译XCFramework
- **核心价值**：预编译的whisper.cpp XCFramework，含CLI和模型下载器
- **二次开发价值**：⭐⭐⭐⭐（最简集成方案）

#### 项目6：NotelyVoice（100+语言支持）
- **地址**：https://github.com/Notely-Voice/NotelyVoice
- **技术栈**：Compose Multiplatform + Whisper AI
- **核心价值**：100%隐私的AI语音转录App，支持100+语言，完全本地处理
- **二次开发价值**：⭐⭐⭐（多语言转录参考，但非Swift原生）

### 7.2 异常处理数据流

```
[录音中断]
    │
    ├─ 来电打断 → AVAudioSession.interruptionNotification
    │   ├─ .began → 暂停录音，保存当前文件
    │   └─ .ended → 恢复录音，继续写入
    │
    ├─ App进入后台 → UIApplication.didEnterBackgroundNotification
    │   └─ 继续录音（后台音频模式已配置）
    │
    ├─ 存储空间不足 → 检查可用空间
    │   ├─ <100MB → 警告提示
    │   └─ <10MB → 停止录音，保存已有内容
    │
    └─ App崩溃 → 文件自动保存
        └─ 下次启动检测未完成录音 → 恢复提示

[转录失败]
    │
    ├─ 模型未下载 → 提示下载
    ├─ 内存不足 → 使用更小模型
    ├─ 音频格式错误 → 重新编码
    └─ 未知错误 → 标记为"转录失败"，允许重试

[同步冲突]
    │
    ├─ 同一录音在两台设备修改
    │   └─ 使用最后修改时间戳，保留最新版本
    └─ 音频文件上传中断
        └─ 断点续传，不重复上传
```

### 8.2 CallKit通话检测服务

```swift
import CallKit
import Combine

final class CallDetectionService: NSObject, ObservableObject {
    @Published var currentCall: CallInfo?
    @Published var isRecording = false
    
    private let callObserver = CXCallObserver()
    private let callController = CXCallController()
    
    struct CallInfo {
        let uuid: UUID
        let isOutgoing: Bool
        let phoneNumber: String?
        let startTime: Date
        var endTime: Date?
        var duration: TimeInterval {
            endTime?.timeIntervalSince(startTime) ?? 0
        }
    }
    
    override init() {
        super.init()
        callObserver.setDelegate(self, queue: nil)
    }
    
    func startObserving() {
        let calls = callObserver.calls
        for call in calls {
            handleCallUpdate(call)
        }
    }
}

extension CallDetectionService: CXCallObserverDelegate {
    func callObserver(
        _ callObserver: CXCallObserver,
        callChanged call: CXCall
    ) {
        handleCallUpdate(call)
    }
    
    private func handleCallUpdate(_ call: CXCall) {
        if call.hasConnected && !call.hasEnded {
            let info = CallInfo(
                uuid: call.uuid,
                isOutgoing: call.isOutgoing,
                phoneNumber: nil,
                startTime: Date()
            )
            DispatchQueue.main.async {
                self.currentCall = info
                self.isRecording = true
            }
            NotificationCenter.default.post(
                name: .callDidStart,
                object: nil,
                userInfo: ["callInfo": info]
            )
        } else if call.hasEnded {
            DispatchQueue.main.async {
                self.currentCall?.endTime = Date()
                self.isRecording = false
            }
            NotificationCenter.default.post(
                name: .callDidEnd,
                object: nil,
                userInfo: ["callInfo": self.currentCall as Any]
            )
        }
    }
}

extension Notification.Name {
    static let callDidStart = Notification.Name("callDidStart")
    static let callDidEnd = Notification.Name("callDidEnd")
```

```swift
        let url = recordingsDirectory

            .appendingPathComponent("\(name).m4a")

            AVNumberOfChannelsKey: 2,
            AVEncoderAudioQualityKey: AVAudioQuality.high.rawValue
        ]
        
        audioFile = try AVAudioFile(forWriting: url, settings: settings)
        
        audioEngine = AVAudioEngine()
        guard let audioEngine = audioEngine else {
            throw RecordingError.engineCreationFailed
        }
        
        let inputNode = audioEngine.inputNode
        let format = inputNode.outputFormat(forBus: 0)
        
        inputNode.installTap(onBus: 0, bufferSize: 1024, format: format) { [weak self] buffer, time in
            try? self?.audioFile?.write(from: buffer)
            self?.updateAudioLevel(buffer: buffer)
        }
        
        try audioEngine.start()
        recordingURL = url
        isRecording = true
        startTimer()
        
        return url
    }
    
    func stopRecording() -> URL? {
        audioEngine?.inputNode.removeTap(onBus: 0)
        audioEngine?.stop()
        audioFile = nil
        isRecording = false
        stopTimer()
        return recordingURL
    }
    
    private func startTimer() {
        timer = Timer.scheduledTimer(withTimeInterval: 0.1, repeats: true) { [weak self] _ in
            self?.recordingDuration += 0.1
        }
    }
    
    private func stopTimer() {
        timer?.invalidate()
        timer = nil
        recordingDuration = 0
    }
    
    private func updateAudioLevel(buffer: AVAudioPCMBuffer) {
        guard let channelData = buffer.floatChannelData?[0] else { return }
        let frameLength = Int(buffer.frameLength)
        var rms: Float = 0
        for i in 0..<frameLength { rms += channelData[i] * channelData[i] }
        rms = sqrt(rms / Float(frameLength))
        DispatchQueue.main.async { self.audioLevel = rms }
    }
}

        case .fileCreationFailed: return "Failed to create audio file"
        }
    }
```

```swift
import Foundation
import AVFoundation

final class TranscriptionEngine: ObservableObject {
    @Published var isTranscribing = false
    @Published var progress: Double = 0
    
    private var whisperContext: OpaquePointer?
    private var currentModel: ModelSize = .base
    
    enum ModelSize: String, CaseIterable {
        case base = "ggml-base.en"
        case small = "ggml-small.en"
        case baseMulti = "ggml-base"
        
        var displayName: String {
            switch self {
            case .base: return "Base English (74MB)"
            case .small: return "Small English (244MB)"
            case .baseMulti: return "Base Multilingual (74MB)"
            }
        }
    }
    
    struct TranscriptSegment: Identifiable {
        let id = UUID()
        let startTime: TimeInterval
        let endTime: TimeInterval
        let text: String
        let speaker: String?
    }
    
    struct TranscriptResult {
        let fullText: String
        let segments: [TranscriptSegment]
        let language: String
        let processingTime: TimeInterval
    }
    
    func loadModel(_ size: ModelSize) async throws {
        let modelURL = try await ensureModelDownloaded(size)
        var params = whisper_context_default_params()
        params.use_gpu = true
        whisperContext = whisper_init_from_file_with_params(modelURL.path, params)
        currentModel = size
    }
    
    func transcribe(audioURL: URL) async throws -> TranscriptResult {
        guard let context = whisperContext else {
            throw TranscriptionError.modelNotLoaded
        }
        
        await MainActor.run { isTranscribing = true }
        let startTime = Date()
        
        let wavURL = try convertToWAV(audioURL)
        
        var params = whisper_full_default_params(WHISPER_SAMPLING_GREEDY)
        params.language = "en"
        params.print_progress = false
        params.print_timestamps = true
        params.single_segment = false
        
        let audioData = try Data(contentsOf: wavURL)
        let frameCount = audioData.count / 2
        var pcmf32 = [Float](repeating: 0, count: frameCount)
        
        pcmf32.withUnsafeMutableBufferPointer { buffer in
            audioData.withUnsafeBytes { rawBuffer in
                let int16Buffer = rawBuffer.bindMemory(to: Int16.self)
                for i in 0..<frameCount {
                    buffer[i] = Float(int16Buffer[i]) / 32768.0
                }
            }
        }
        
        try pcmf32.withUnsafeBufferPointer { buffer in
            if whisper_full(context, params, buffer.baseAddress, Int32(buffer.count)) != 0 {
                throw TranscriptionError.transcriptionFailed
            }
        }
        
        let segmentCount = whisper_full_n_segments(context)
        var segments: [TranscriptSegment] = []
        
        for i in 0..<segmentCount {
            let text = String(cString: whisper_full_get_segment_text(context, i))
            let start = Double(whisper_full_get_segment_t0(context, i)) / 100.0
            let end = Double(whisper_full_get_segment_t1(context, i)) / 100.0
            segments.append(TranscriptSegment(
                startTime: start,
                endTime: end,
                text: text.trimmingCharacters(in: .whitespacesAndNewlines),
                speaker: nil
            ))
        }
        
        let fullText = segments.map(\.text).joined(separator: " ")
        let processingTime = Date().timeIntervalSince(startTime)
        
        await MainActor.run { isTranscribing = false }
        
        return TranscriptResult(
            fullText: fullText,
            segments: segments,
            language: "en",
            processingTime: processingTime
        )
    }
    
    private func convertToWAV(_ url: URL) throws -> URL {
        let asset = AVAsset(url: url)
        let outputURL = url.deletingPathExtension().appendingPathExtension("wav")
        guard let exportSession = AVAssetExportSession(
            asset: asset,
            presetName: AVAssetExportPresetPassthrough
        ) else { throw TranscriptionError.conversionFailed }
        
        exportSession.outputURL = outputURL
        exportSession.outputFileType = .wav
        let semaphore = DispatchSemaphore(value: 0)
        exportSession.exportAsynchronously { semaphore.signal() }
        semaphore.wait()
        return outputURL
    }
    
    private func ensureModelDownloaded(_ size: ModelSize) async throws -> URL {
        let modelsDir = FileManager.default.urls(
            for: .applicationSupportDirectory, in: .userDomainMask
        )[0].appendingPathComponent("WhisperModels")
        try FileManager.default.createDirectory(at: modelsDir, withIntermediateDirectories: true)
        let modelFile = modelsDir.appendingPathComponent("\(size.rawValue).bin")
        
        if FileManager.default.fileExists(atPath: modelFile.path) { return modelFile }
        
        guard let downloadURL = URL(string: "https://huggingface.co/ggerganov/whisper.cpp/resolve/main/\(size.rawValue).bin") else {
            throw TranscriptionError.modelDownloadFailed
        }
        let (tempURL, _) = try await URLSession.shared.download(from: downloadURL)
        try FileManager.default.moveItem(at: tempURL, to: modelFile)
        return modelFile
    }
}

enum TranscriptionError: LocalizedError {
    case modelNotLoaded
    case conversionFailed
    case modelDownloadFailed
    case transcriptionFailed
    var errorDescription: String? {
        switch self {
        case .modelNotLoaded: return "AI model not loaded. Please wait for download."
        case .conversionFailed: return "Failed to convert audio format."
        case .modelDownloadFailed: return "Failed to download AI model."
        case .transcriptionFailed: return "Transcription failed. Please try again."
        }
    }
}
```

### 8.5 AI摘要生成引擎

```swift
import Foundation

final class SummaryEngine: ObservableObject {
    @Published var isGenerating = false
    
    struct ActionItem: Identifiable, Codable {
        let id = UUID()
        let text: String
        let deadline: String?
        let assignee: String?
        var isCompleted = false
    }
    
    struct CallSummary: Codable {
        let overview: String
        let keyPoints: [String]
        let actionItems: [ActionItem]
        let sentiment: Sentiment
        let topics: [String]
    }
    
    enum Sentiment: String, Codable {
        case positive = "Positive"
        case neutral = "Neutral"
        case negative = "Negative"
        case mixed = "Mixed"
    }
    
    func generateSummary(transcript: String) async throws -> CallSummary {
        await MainActor.run { isGenerating = true }
        defer { Task { @MainActor in isGenerating = false } }
        
        if #available(iOS 18.0, *) {
            return try await generateWithOnDeviceLLM(transcript: transcript)
        } else {
            return try await generateWithLocalRules(transcript: transcript)
        }
    }
    
    @available(iOS 18.0, *)
    private func generateWithOnDeviceLLM(transcript: String) async throws -> CallSummary {
        let prompt = """
        Analyze this phone call transcript and provide:
        1. A one-sentence overview
        2. 3-5 key points
        3. Action items with deadlines if mentioned
        4. Overall sentiment (Positive/Neutral/Negative/Mixed)
        5. Main topics discussed
        
        Transcript:
        \(transcript)
        
        Respond in JSON format:
        {"overview":"...","keyPoints":["..."],"actionItems":[{"text":"...","deadline":"...","assignee":"..."}],"sentiment":"Positive|Neutral|Negative|Mixed","topics":["..."]}
        """
        let response = try await AppleIntelligenceService.generate(prompt: prompt)
        return try JSONDecoder().decode(CallSummary.self, from: response.data(using: .utf8)!)
    }
    
    private func generateWithLocalRules(transcript: String) async throws -> CallSummary {
        let sentences = transcript.components(separatedBy: ". ").filter { $0.count > 20 }
        let overview = String(sentences.first ?? "No summary available.").prefix(200).description
        let keyPoints = Array(sentences.prefix(5))
        
        let actionPatterns = [
            "need to", "should", "will", "must", "have to",
            "let's", "make sure", "don't forget", "follow up",
            "send", "schedule", "call back", "review", "confirm"
        ]
        let actionItems = sentences
            .filter { s in actionPatterns.contains { s.lowercased().contains($0) } }
            .prefix(5)
            .map { ActionItem(text: $0, deadline: nil, assignee: nil) }
        
        let positiveWords = ["great", "excellent", "agree", "confirm", "happy", "love", "perfect"]
        let negativeWords = ["problem", "issue", "concern", "disagree", "unfortunately", "sorry", "wrong"]
        let lower = transcript.lowercased()
        let posCount = positiveWords.reduce(0) { $0 + (lower.contains($1) ? 1 : 0) }
        let negCount = negativeWords.reduce(0) { $0 + (lower.contains($1) ? 1 : 0) }
        
        let sentiment: Sentiment
        if posCount > negCount + 2 { sentiment = .positive }
        else if negCount > posCount + 2 { sentiment = .negative }
        else if posCount > 0 && negCount > 0 { sentiment = .mixed }
        else { sentiment = .neutral }
        
        return CallSummary(
            overview: overview,
            keyPoints: keyPoints,
            actionItems: Array(actionItems),
            sentiment: sentiment,
            topics: extractTopics(from: transcript)
        )
    }
    
    private func extractTopics(from text: String) -> [String] {
        let keywords = ["budget","project","meeting","deadline","client","contract","proposal","review","schedule","team"]
        return keywords.filter { text.lowercased().contains($0) }
    }
}
```

- 限制条款：不含未来付费API服务，保护长期利益

#### 免费版为什么提供5次转录/月？

- 5次/月足够轻度用户体验AI转录价值
- 超过5次后显示升级提示："You've used 5/5 free transcriptions this month. Upgrade to Pro for unlimited."
- 转化心理学：用户尝到甜头后更愿意付费
- 录音功能完全免费，确保基础价值不受限
